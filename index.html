neovim.tips API

Get a Random Tip

Endpoint: /api/random
Method: GET

Returns a random tip from the collection of neovim tips.

curl -X GET https://www.neovim.tips/api/random

Get a Specific Tip by ID

Endpoint: /api/{id}
Method: GET

Fetches a specific tip based on its unique identifier (ID).

curl -X GET https://www.neovim.tips/api/22
